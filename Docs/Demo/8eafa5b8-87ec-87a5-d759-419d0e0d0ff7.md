# ISimpleAccess(*TDbConnection*, *TDbTransaction*, *TDbCommand*, *TDataParameter*, *TDbDataReader*, *TParameterBuilder*).ExecuteScalar(*T*) Method (*TDbTransaction*, String, CommandType, *TDataParameter*[])
